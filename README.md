# 🏦 Bank Management System

A Bank Management System developed using Python, MySQL, and Pandas.

## 📌 Features
- Create Account
- View Account
- Update Account
- Delete Account
- Deposit Money
- Withdraw Money
- Store customer data in MySQL

## 🛠 Technologies Used
- Python
- MySQL
- Pandas

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   ```bash
   pip install mysql-connector-python pandas
   ```
3. Create the MySQL database.
4. Run:
   ```bash
   python Bank-Management-System.py
   ```

## 📷 Output
The project is menu-driven and allows users to perform banking operations using a MySQL database.
